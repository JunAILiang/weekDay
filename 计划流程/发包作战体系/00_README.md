## 这套体系是什么

解决三个问题：
1. **不知道去哪找** — 本文件是入口，所有内容从这里导航
2. **不知道怎么做** — 去 `02_全流程SOP.md`
3. **遇到问题不知道怎么处理** — 去 `04_问题知识库.md`

## 当前核心目标

| 指标 | 当前 | 目标 |
|------|------|------|
| 月产包数 | ~14 包/月（2人） | 30 包/月（3人） |
| 过审率 | ~83% | ≥ 90% |
| 可用账号库存 | - | 常备 ≥ 10 个 |
| 卡审率 | 30%+ | < 10% |
