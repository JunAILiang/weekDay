# 机器人 Call 呼叫漏斗报表需求

## 一、需求背景

统计假机器人 Call 策略的核心转化效果，评估从来电展示到接听到观看完成的完整用户漏斗。

## 二、数据源

| 事件名 | 说明 | 关键字段 |
|-------|------|---------|
| `robot_incoming_displayed` | 来电弹窗展示 | `user_id`, `robot_anchor_id` |
| `robot_call_connected` | 用户点击接听 | `user_id`, `robot_anchor_id` |
| `robot_call_missed` | 用户拒接/超时 | `user_id`, `robot_anchor_id` |
| `robot_call_ended` | 通话结束 | `user_id`, `robot_anchor_id`, `end_reason`, `call_duration` |

## 三、筛选条件

- 时间范围：按日期筛选
- 平台：iOS / Android（可选）
- 机器人ID：按 `robot_anchor_id` 筛选（可选）

## 四、核心指标

| 指标名称 | 说明 |
|---------|------|
| 来电展示 UV | 收到来电的用户数 |
| 接听 UV | 接听的用户数 |
| 接听率 | 接听用户占来电展示用户的比例 |
| 拒接 UV | 拒接/超时的用户数 |
| 拒接率 | 拒接用户占来电展示用户的比例 |
| 完播 UV | 看完60秒的用户数（`end_reason='video_end'`） |
| 完播率 | 完播用户占接听用户的比例 |
| 平均观看时长 | 用户平均观看秒数 |

## 五、统计维度

- 按日期（必选）
- 按平台（必选）：iOS / Android
- 按机器人ID（可选）

## 七、漏斗示例

```
来电展示: 1,000 UV
    ↓
接听: 500 UV (接听率 50%)
    ↓
完播: 300 UV (完播率 60%)
平均观看: 45秒
```
