# 第69天：询问之确定责权利学习总结

## 核心洞察

- **责权利三位一体的核心框架**：在向上管理中清晰地界定责任（责）权限（权）和利益（利）是推动工作避免扯皮确保目标达成的基石，这三者是一个动态的整体需要主动沟通和协商来确定。谁负责谁拍板谁决策责任权力利益是三位一体的绝对不能错配第69天，一旦责任错配会导致项目方向混乱最终结果无人说得清。本质是第11天向谁汇报的"目标管理三件套"（目标-动作-效果）+第56天抓情绪的"价值交换意识"（成年人只谈交换），责权利的明确是资源争取的前提
- **权不够责上找利上让的核心策略**：权力=调动资源的能力第11天向谁汇报，如果领导只给你责任不给你授权（调动资源的能力）工作会很难开展因为绩效考核奖金分配人员调整等资源都不在你手上事情就干不好。权不够责上找利上让第69天，核心话术"事情干成了算你们的没干成算我的"收益分析：没干成你没损失什么干成了你赚了能力（硬通货）+信誉（下次领导会更信你）+人脉第14天。本质是第56天抓情绪的"利上让交换空间Space"+第64天不麻木的"主动担当要资源"
- **权力三分类与软话办硬事**：人权（岗位组织协调）+财权（钱绩效）+事权（决策时间）第69天，获取话术"这个事我就按照咱们确定去推进了""麻烦您和XX打个招呼之后我调配资源的时候比较方便""推进的时候我需要XX和XX来配合我有问题的时候我来找您"。软话办硬事：什么是软话就是拿出你对上级的态度去跟平级下级去沟通第69天。本质是第11天向谁汇报的"资源争取"（向谁汇报决定了你能获取的资源）+第18天独立思考的"沟通能力"（软话办硬事）
- **利不够利里找和量不够集中给**：利不够利里找要去拿未来的天花板来获得共识和共赢（案例：月薪5000想涨到6000老板不给但如果说帮公司盈利100万你分我30万就是用未来的大利益争取当下的投入第20天司马迁三阶段）。要想赏多那就罚多对赌模式高回报必须与高风险绑定本质是穿一条裤子第69天。量不够集中给资源（如奖金）永远是不够的关键在于分配方式与其雨露均沾（无效激励）不如集中奖励给做出核心贡献的人第69天。本质是第56天抓情绪的"价值交换"（用承担风险交换升职加薪Promotion）+第52天复合优势的"指数增长思维"

## 破局心法

**从"责权利错配扯皮"到"三位一体推进"的向上管理模式**：不再被动接受有责无权的困境、不再雨露均沾做无效激励，而是建立"谁负责谁拍板+权不够责上找利上让+权力三分类软话办硬事+利不够利里找集中给"的四位一体责权利系统，让向上管理从"扯皮推诿"升级为"高效协作"，用责权利的明确赢得资源支持和团队配合。

## 四大核心行动

### 行动一：建立谁负责谁拍板的责任明确系统
**实施方案**：
1. 核心原则：谁负责谁拍板谁决策责任权力利益是三位一体的绝对不能错配第69天
2. 话术："老大关于XX这件事之后我找您汇报您看可以吗"（办公室情商的体现第59天观察偏好）
3. 提问技巧：开放型问题"您觉得这个谁负责比较合适"vs封闭型问题"我觉得这个事让XX负责更好您看呢"（推荐第66天提问）
4. 案例应用：抖音号的预算谁负责假设要做到200万的营收如果是张三负责那么在营收这件事上哪怕我是领导也要听张三的这就是谁负责什么就听谁的第69天
5. 借势：你有能力领导有资源借领导的势（资源和影响力）来办自己的事前提是责任清晰第11天向谁汇报

### 行动二：建立权不够责上找利上让的交换系统
**实施方案**：
1. 核心认知：权=调动资源的能力权责困境=领导只给责任不给授权工作难以开展第11天向谁汇报
2. 策略：权不够责上找利上让第69天核心话术"事情干成了算你们的没干成算我的"
3. 收益分析：没干成你没损失什么干成了你赚了能力（硬通货第27天刻意练习）+信誉（下次领导会更信你第61天值得跟随）+人脉（第14天人际关系）
4. 用第56天抓情绪的"利上让交换空间Space"，主动谦让利益换取和平的工作空间（挑拨→谦让利益→交换Space）
5. 用第64天不麻木的"主动担当要资源"，从"我不能"到"我能但我需要XX资源"（强者语言）

### 行动三：建立权力三分类的软话办硬事系统
**实施方案**：
1. 权力三分类：人权（岗位组织协调）+财权（钱绩效）+事权（决策时间）第69天
2. 要人权话术："麻烦您和XX打个招呼之后我调配资源的时候比较方便""推进的时候我需要XX和XX来配合我有问题的时候我来找您""完成这个目标我可能还需要XX和XX"
3. 要事权话术："这个事我就按照咱们确定去推进了"（确认责任与权力）
4. 软话办硬事：拿出你对上级的态度去跟平级下级去沟通第69天（本质是第18天独立思考的"沟通能力"）
5. 用第11天向谁汇报，主动清晰地向上级要资源是核心能力第64天（表明你对目标有承诺对路径有清晰思考）

### 行动四：建立利不够利里找的对赌集中系统
**实施方案**：
1. 利不够利里找：要去拿未来的天花板来获得共识和共赢第69天（案例：月薪5000→帮公司盈利100万分我30万）
2. 用第20天副业思维的"司马迁三阶段"，从无财作力（trade时间换钱）到少有斗智（trade知识换钱）到既饶争时（trade钱换机会）
3. 要想赏多那就罚多对赌模式：高回报必须与高风险绑定本质是穿一条裤子第69天（案例：目标200万做到分50万没做到个人补50万）
4. 量不够集中给：资源（如奖金）永远是不够的关键在于分配方式与其雨露均沾（无效激励）不如集中奖励给做出核心贡献的人第69天
5. 沟通话术："我把绩效逻辑跟您过一下如果没问题我就和人力去确认了""能不能在奖金机制上再增加一些力度"

## 与既往整合

- **对齐第11天向谁汇报**：责权利的核心是向谁汇报的资源争取。谁负责谁拍板=向谁汇报（向谁汇报决定了你能获取的资源），权=调动资源的能力（人权+财权+事权），利=资源分配方式（集中给核心贡献者），责权利明确是资源争取的前提
- **对齐第56天抓情绪**：责权利的交换逻辑是抓情绪的价值交换应用。权不够利上让=挑拨→谦让利益→交换Space，要想赏多那就罚多=用承担风险交换升职加薪Promotion，成年人只谈交换第56天ARTTSTP模型
- **对齐第64天不麻木**：权不够责上找利上让的本质是不麻木的主动担当。从"我不能"（弱者自我设限）到"我能但我需要XX资源"（强者担当），主动清晰地向上级要资源是核心能力第64天，目标刚性方法弹性
- **对齐第66天提问能力**：确定责权利的前置是提问能力。封闭型问题"我觉得这个事让XX负责更好您看呢"（带着作业对答案），开放型问题"您觉得这个谁负责比较合适"（直接索要答案），提问技巧是责权利明确的沟通基础
- **对齐第68天SMART**：责权利是SMART目标的资源保障。S具体化+M可衡量+A可实现（需要权力支持）+R相关性+T时限性，确定责权利=确保SMART目标有资源支持（人权+财权+事权），没有权力的目标无法执行

> 最终公式：**责权利 = 谁负责谁拍板 × 权不够责上找利上让 × 权力三分类软话办硬事 × 利不够利里找集中给 = 三位一体高效协作**
