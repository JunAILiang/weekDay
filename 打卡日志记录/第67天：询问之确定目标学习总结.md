# 第67天：询问之确定目标学习总结

## 核心洞察

- **带着作业对答案是向上管理核心要义**：这是向上管理最核心的原则也是区分员工能力层级的关键，帮领导省时间通过主动思考和准备方案节省领导定义和解释任务的时间体现自身价值。本质是第66天提问能力的"带着思考去请求指点"+第18天独立思考的"主动构建能力"，从被动等待指令升级为主动定义目标
- **员工三个档次的能力分层**：一档员工带着作业对答案（唯一能够持续成长和发展的模式主动思考将领导模糊意图转化为具体方案然后请求确认和校准第66天提问）、二档员工问完答案做作业（缺乏主动构建解决方案的能力需要等待明确的指令才开始行动能力成长受限第18天依赖期）、三档员工不做作业对答案（纯粹的伸手党仅仅是一个执行者可替代性极高把思考的责任完全推给了领导）。如果你只是一个被动的执行者就缺少了最关键的构建能力这意味着换任何一个人来都能做你的核心价值无法体现第52天复合优势前20%
- **课题分离的心态修炼**：很多人在职场中没有做到课题分离，什么是课题分离：领导批评你是他的事这是他的课题而我要把这件事弄清楚干明白这是我的课题，不要让别人的情绪和行为干扰到你自己核心要完成的任务。告诫：不要总是在职场中用抱怨负能量等方式来表现自己的脆弱第8天情绪管理的依赖期。本质是第62天不评判的"远离立场思考"+第53天倾听的"心态抽离"，将领导作为抽象词来学习原则而非代入具体领导的情绪
- **将模糊目标具体化的SMART应用**：当目标大而空的时候首先要展现积极的态度"没问题夏老师这是我来搞定"第56天抓需求的态度工具，然后帮助领导明确目标的S（Specific具体化）"如果要达到拉起来这个目标我们要做36节课下个月1号就开始动起来您觉得呢"，再带着作业去对答案"假如我们把开班仪式做了把小组组长选了反馈表也设计好了您看这个是不是达到了您拉起来的目标"。本质是第1天SMART目标的向上管理应用+第11天目标拆解的"大目标→子任务+验证标准"

## 破局心法

**从"被动执行指令"到"主动定义目标"的向上管理模式**：不再等待领导给出明确指令、不再做伸手党缺少构建能力，而是建立"带着作业对答案+一档员工成长+课题分离心态+SMART具体化"的四位一体确定目标系统，让向上管理从"三档执行者"升级为"一档构建者"，用主动定义目标能力赢得持续成长空间。

## 四大核心行动

### 行动一：建立带着作业对答案的准备系统
**实施方案**：
1. 用第66天提问能力，提问前先思考将领导模糊意图转化为具体方案（带着作业）
2. 帮领导省时间：通过主动思考和准备方案节省领导定义和解释任务的时间第54天场合媒介一对一成本高
3. 用第18天独立思考，主动构建解决方案的能力（这是最关键的构建能力）
4. 千万不要像个新手一样去问"我们应该怎么做反馈表"（直接问答案定义目标的责任还在领导身上）

### 行动二：建立一档员工的成长路径
**实施方案**：
1. 一档员工：用第66天提问+第1天SMART目标，带着作业对答案（唯一能够持续成长和发展的模式）
2. 避免二档员工陷阱：不要问完答案做作业（缺乏主动构建能力等待明确指令第18天依赖期）
3. 避免三档员工陷阱：不要不做作业对答案（伸手党执行者可替代性极高缺少构建能力）
4. 用第52天复合优势，将"主动构建目标能力"作为前20%的核心能力持续强化第27天刻意练习

### 行动三：建立课题分离的心态系统
**实施方案**：
1. 用第62天不评判的"远离立场思考"，领导批评你是他的课题（他的情绪）不是你的课题
2. 用第53天倾听的"心态抽离"，不要让别人的情绪和行为干扰到你自己核心要完成的任务
3. 你的课题是：把这件事弄清楚干明白（聚焦解决问题而非被情绪干扰第62天事实层面）
4. 用第8天情绪管理，不要总是在职场中用抱怨负能量等方式来表现自己的脆弱（从依赖期进入独立期）

### 行动四：建立SMART具体化的目标翻译系统
**实施方案**：
1. 用第1天SMART目标，将领导的模糊指令翻译成具体目标（S=Specific具体化）
2. 第一步展现态度："没问题夏老师这是我来搞定"第56天抓需求的态度工具（当目标大而空时首先要展现积极态度）
3. 第二步具体化："如果要达到拉起来这个目标我们要做36节课下个月1号就开始动起来您觉得呢"（帮助领导明确S）
4. 第三步带着作业对答案："假如我们把开班仪式做了把小组组长选了反馈表也设计好了您看这个是不是达到了您拉起来的目标"（用第11天目标拆解，大目标→子任务+验证标准）

## 与既往整合

- **对齐第1天SMART目标**：确定目标的本质是SMART目标的向上管理应用。将领导模糊指令（"拉起来"）翻译成Specific（36节课+1号开始）+Measurable（开班仪式+小组组长+反馈表）+Time-bound（下个月），从模糊大而空升级为具体可执行
- **对齐第18天独立思考**：一档员工vs三档员工的本质是独立思考能力。一档=主动构建解决方案（独立思考），三档=伸手党执行者（依赖指令），带着作业对答案=独立思考能力的体现，缺少构建能力=可替代性极高
- **对齐第52天复合优势**：主动构建目标能力是前20%的核心能力。用足够的努力（刻意练习第27天）将"带着作业对答案"能力练到前20%，叠加其他前20%能力（如心理表征第36天）形成前4%的复合优势
- **对齐第62天不评判**：课题分离的心态修炼是不评判的应用。领导批评你=立场性言论（他的课题），你要把事情弄清楚=事实层面解决问题（你的课题），远离立场思考聚焦事实，不让别人的情绪干扰你的任务
- **对齐第66天提问能力**：确定目标是提问能力的具体应用场景。第66天提问=带着作业对答案（先思考后提问验证），第67天确定目标=将模糊意图转化为具体方案（SMART具体化），两者构成目标确定的完整闭环（提问验证→目标确定）

> 最终公式：**确定目标 = 带着作业对答案 × 一档员工成长 × 课题分离心态 × SMART具体化 = 主动构建能力持续成长**
